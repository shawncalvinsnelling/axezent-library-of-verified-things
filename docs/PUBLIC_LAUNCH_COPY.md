# Public Launch Copy

## Short announcement

Axezent AI is launching the Library of Verified Things: a receipt-backed verification layer for AI, software, and research claims.

The rule is simple:

```text
No claim without a receipt.
```

The public verifier checks finite JSON receipts and returns ACCEPT, REJECT, or INCOMPLETE under a declared truth boundary.
