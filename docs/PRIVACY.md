# Privacy — Plain Language Starter

This starter text is not legal advice.

The static website runs verification locally in the browser. The included local API runs on your machine.

For a hosted version, the company should publish a full privacy policy before collecting customer data.

Recommended hosted privacy principles:

- collect the minimum data needed;
- do not sell receipt data;
- allow customers to delete private artifacts;
- keep private audit vaults separate from public samples;
- never publish private claims without permission.
